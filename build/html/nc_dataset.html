ncobj.nc_dataset¶

Module for converting ncobj data to and from NetCDF Datasets.

ncobj.nc_dataset.read(dataset)¶

Read a dataset from a netCDF file.

Args:

  • dataset (netCDF4.Dataset):

    An open readable file to scan for information.

Returns:
A ncobj.Group representing the entire dataset.

Note

The returned data retains references to the netCDF4 dataset, which therefore must remain open while any of that data may still be needed. (For this reason, we don’t yet support read from a filepath string).

ncobj.nc_dataset.write(dataset, group)¶

Write a dataset to a netCDF file.

Args:

  • dataset (netCDF4.Dataset or string):

    An open writeable file, or a path string to create one. If a file was created, it is closed again afterwards.

  • group (ncobj.Group):

    Data to write. Note that this is passed to ncobj.grouping.complete(), which will usually modify it.

Note

Writing data into an existing file can obviously cause problems, but will equally obviously work fine in specific cases.

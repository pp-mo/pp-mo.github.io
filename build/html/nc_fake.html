ncobj.nc_fake¶

Module for wrapping an ncobj.Group to make it as far as possible appear as if it were a netCDF4.Dataset. This is for read-only purposes : all dataset-changing methods are missing.

At present this is focussed on delivering the ability to load these quasi-datasets into the Iris project (https://github.com/SciTools/iris) : The emulation of any behaviours not used there are currently uncertain.

ncobj.nc_fake.fake_nc4python_dataset(ncobj_group)¶

Make a wrapper around an ncobj.Group object to emulate a :class:`netCDF4.Dataset’.

The resulting GroupMimic supports the essential properties of a read-mode :class:`netCDF4.Dataset’, enabling an arbitrary netcdf data structure in memory to be “read” as if it were a file (i.e. without writing it to disk).

In particular, variable data access is delegated to the original, underlying ncobj.Group object : This provides deferred, sectional data access on request, in the usual way, avoiding the need to read in all the variable data.
